Projeto de Jogo Digital desenvolvido para a disciplina Introdução à Programação para Jogos da UFABC em 2021.2.

‘Fale Comigo’ é um jogo de plataforma 2D para PC com foco na narrativa. 
Com seu visual em Pixel Art, ele conta a história de Michael, um garoto de 15 anos que após se mudar para uma nova cidade, parte em busca de elementos que formem uma conexão com as pessoas de sua nova escola, fazendo com que
ele possa se comunicar com elas. O jogador irá ir atrás desses elementos por meio de um desafio de plataforma vertical, e irá acompanhar essa envolvente história pessoal.
