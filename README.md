# Fale-Comigo

Projeto de Jogo Digital desenvolvido para a disciplina **Introdução à Programação para Jogos** da UFABC no período letivo de 2021.2.

## Sobre o projeto

**Fale-Comigo** é um jogo de plataforma 2D para PC com foco em narrativa. O projeto acompanha Michael, um garoto de 15 anos que, após se mudar para uma nova cidade, precisa encontrar elementos que o ajudem a criar conexões com as pessoas de sua nova escola e, assim, conseguir se comunicar com elas.

Com visual em Pixel Art e progressão baseada em desafio de plataforma vertical, o jogo combina movimentação, exploração e desenvolvimento narrativo em uma experiência centrada no crescimento pessoal do protagonista.

## Contexto acadêmico

Este projeto foi desenvolvido como trabalho da disciplina de graduação **Introdução à Programação para Jogos** da Universidade Federal do ABC (UFABC), em 2021.2. O objetivo foi aplicar conceitos introdutórios de desenvolvimento de jogos digitais, incluindo estruturação de cenas, lógica de gameplay, organização de assets e implementação de mecânicas em ambiente Unity.

## Premissa

Michael acabou de chegar a uma nova cidade. Em meio à dificuldade de adaptação e ao desafio de se aproximar de novas pessoas, ele embarca em uma jornada simbólica para encontrar os elementos necessários para construir vínculos e se expressar. A narrativa se desenvolve junto à progressão do jogador pelos obstáculos do cenário.

## Gênero e proposta

- Plataforma 2D.
- Foco narrativo.
- Progressão vertical.
- Estética em Pixel Art.
- Desenvolvido para PC.
